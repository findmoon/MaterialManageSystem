## 严重问题（必须修复）

### 1. 密码安全漏洞
**文件**: `MaterialManageSystem.Api/Controllers/AuthController.cs`
**问题**: `VerifyPassword` 方法使用明文比较密码
```csharp
private bool VerifyPassword(string password, string passwordHash)
{
    // Simple verification - in production, use proper password hashing
    return password == passwordHash;
}
```
**风险**: 密码明文存储，严重安全隐患
**修复方案**: 使用 BCrypt.Net-Next 或类似库进行密码哈希
```csharp
// 安装包: BCrypt.Net-Next
private bool VerifyPassword(string password, string passwordHash)
{
    return BCrypt.Net.BCrypt.Verify(password, passwordHash);
}

// 注册时:
user.PasswordHash = BCrypt.Net.BCrypt.HashPassword(password);
```

### 2. 料卷报废状态错误
**文件**: `MaterialManageSystem.Core/Enums/Enums.cs`
**问题**: `ReelStatus` 枚举缺少 `Scrapped`（已报废）状态
```csharp
public enum ReelStatus : int
{
    InStock = 0,    // 在库
    OutStock = 1,   // 出库
    Online = 2      // 在线
    // 缺少: Scrapped = 3  // 已报废
}
```
**影响**: `ReelsController.Scrap` 方法将报废料卷状态设置为 `InStock`，导致逻辑错误

**修复方案**:
```csharp
public enum ReelStatus : int
{
    InStock = 0,
    OutStock = 1,
    Online = 2,
    Scrapped = 3    // 添加已报废状态
}
```

同时修复 `ReelsController.Scrap` 方法：
```csharp
reel.Status = ReelStatus.Scrapped;  // 改为已报废状态
```

        // ...

### 7. CORS配置不安全
**文件**: `MaterialManageSystem.Api/Program.cs`
**问题**: 允许所有来源
```csharp
policy.AllowAnyOrigin()
      .AllowAnyMethod()
      .AllowAnyHeader();
```
**修复方案**: 生产环境限制具体域名
```csharp
builder.Services.AddCors(options =>
{
    options.AddPolicy("Production", policy =>
    {
        policy.WithOrigins("https://yourdomain.com")
              .AllowAnyMethod()
              .AllowAnyHeader()
              .AllowCredentials();
    });
    
    options.AddPolicy("Development", policy =>
    {
        policy.AllowAnyOrigin()
              .AllowAnyMethod()
              .AllowAnyHeader();
    });
});

// 使用时:
if (app.Environment.IsProduction())
    app.UseCors("Production");
else
    app.UseCors("Development");
```

### 8. 缺少全局异常处理
**文件**: `MaterialManageSystem.Api/Program.cs`
**问题**: 没有配置全局异常处理中间件

**修复方案**: 添加异常处理中间件
```csharp
// 创建中间件
public class ExceptionMiddleware
{
    public async Task InvokeAsync(HttpContext context, RequestDelegate next)
    {
        try
        {
            await next(context);
        }
        catch (Exception ex)
        {
            _logger.LogError(ex, "发生未处理的异常");
            context.Response.StatusCode = 500;
            await context.Response.WriteAsJsonAsync(new
            {
                Code = 500,
                Message = "服务器内部错误"
            });
        }
    }
}

// 注册
app.UseMiddleware<ExceptionMiddleware>();
```
